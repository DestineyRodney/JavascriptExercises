Variables

Ex: var name = "codeup"; [String type assigned at runtime]

Strings <---> Numbers

.toString() converts number to string
.toFixed(characters returned) converts to number with characters after decimal
Number() converts to number or returns NaN
isNaN() returns true/false based on if value isNaN if it is a number returns false and not a number returns true

Examples:
var x = 3.14;
x.toString(); // returns "3.14"
x.toFixed(3); // returns 3.140
Number('123'); // returns 123
Number('abs'); // returns NaN
isNaN('123'); // returns false
isNaN('abc'); // returns true
isNaN(Number('abc'); // returns true

Loops

While loop
: while this is true execute this code (evaluated at the start)

Do While Loop
:Condition is evaluated at the end so it will run at least once

}while (condition) { //body};

For Loop
for (/*initialization*/; /*condition*/; /*increment*/) {
// body }
